# Origami WEB
This is a website which simulates OriginOS, but better!

# Design
The UI is completely changed and everything is round for a cleaner and more simplistic look, whilst still having functionality.

# Feautres
The features are pretty much the same.

# Q&A
1. **Is this project on track with OriginWEB?**
It is not, because it's a seperate project rather than a fork of OriginWEB.
2. **Will this project be constantly updated?**
Possibly, although not in track with OriginWEB.
3. **Can I create forks of it?**
Yes, as long as you change something.
